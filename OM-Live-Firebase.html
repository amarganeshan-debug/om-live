Step 01
Create a free Firebase project

Go to console.firebase.google.com → Add project → give it any name (e.g. om-live) → skip Google Analytics → Create. Then click Build → Realtime Database → Create database → Start in test mode.

Step 02
Get your config keys

In Firebase console → Project settings (gear icon) → Your apps → Add web app → Register → copy the firebaseConfig object. Paste the values into the fields below.
